# Portfolio Website - Andrej Folta

Modern, minimalist portfolio website showcasing my frontend development skills and experience. Built with React, TypeScript, and Tailwind CSS, featuring smooth animations and parallax effects.

## Live Demo

[View Live Site](https://folty7.github.io/)

## Tech Stack

### Core
- **React 19** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Build tool and dev server
- **Tailwind CSS v4** - Utility-first styling with new PostCSS plugin

### Animation & Effects
- **Framer Motion** - React animation library
- **Lenis** - Smooth scroll implementation
- **react-fast-marquee** - Marquee text effects
